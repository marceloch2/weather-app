# weather-vue-app

IMPORTANT - [First allow temporary access with](https://cors-anywhere.herokuapp.com/corsdemo)

## Project setup
```
git checkout develop
yarn install
```

### Lints and fixes files
```
yarn run lint
```

#### Run your end-to-end tests
```
yarn run test:e2e
```

##### Run your unit tests
```
yarn run test:unit
```

###### Compiles and hot-reloads for development
```
yarn run serve
```
